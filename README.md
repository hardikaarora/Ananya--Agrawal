# Ananya Agrawal — Personal About Me Site

A modern, minimal, responsive personal website template designed for a BBA student interested in business and entrepreneurship.

## Features

- Clean, pastel-themed design with soft shadows and rounded cards
- Smooth scrolling navigation and subtle hover animations
- Sections for About, Interests, Projects, Skills, Fun Facts, and Contact
- Mobile-friendly responsive layout

## Getting Started

### Open locally
1. Open this folder in your browser (double-click `index.html`) or run a quick local server.

### Optional: Run a local server (recommended)
If you have Python installed, run:

```bash
python3 -m http.server
```

Then open `http://localhost:8000` in your browser.

## Customize
- Replace the placeholder profile photo in `index.html` with a real image.
- Update contact details and social links.
- Add your own project images and descriptions.
